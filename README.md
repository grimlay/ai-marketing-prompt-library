# 🧠 AI Marketing Prompt Library

A curated collection of Claude prompts for marketing professionals — built from real campaign work across email, social media, content/SEO, and paid ads.

Each prompt includes usage notes explaining **when to use it**, **what to customise**, and **what good output looks like**.

---

## Why this exists

Most AI prompt guides are generic. This library is opinionated — built from hands-on experience running campaigns for B2B events and content platforms. The prompts here are ones I actually use, refined over time to produce outputs that require minimal editing.

---

## How to use this library

1. Copy the prompt into [Claude](https://claude.ai)
2. Replace anything in `[square brackets]` with your specifics
3. Review the **Usage Notes** section in each file before running

All prompts are optimised for **Claude (Sonnet or above)**. They may work in other models but have not been tested.
