# ✍️ Content & SEO Prompts

---

## 1. Blog Post Brief Generator

**The Prompt**
```
Create a detailed brief for a blog post on the following topic:

Topic: [e.g. how open banking is enabling SME lending in Southeast Asia]
Target audience: [e.g. fintech product managers and banking strategists]
Primary keyword: [e.g. open banking Southeast Asia]
Secondary keywords: [e.g. SME lending, embedded finance, API banking]
Desired word count: [e.g. 1,200 words]
Angle/hook: [e.g. most coverage focuses on consumer banking — this piece focuses on SMEs]

Brief should include: suggested headline, subheadings, key points per section, recommended sources or data to reference, and meta description.
```

**Usage Notes**
- Use this to brief freelancers or to structure your own writing — it saves 30–45 minutes of planning
- The "angle/hook" field is the most important; without it, briefs come out generic
- Always review the meta description — Claude often writes it too long (keep under 155 characters)

## 2. Meta Description Writer

**The Prompt**
```
Write 5 meta description variants for the following page:

Page title: [e.g. Money20/20 Asia 2026 — Speaker Lineup]
Page content summary: [2–3 sentences about what the page covers]
Primary keyword: [e.g. fintech conference Asia 2026]
Character limit: 150–155 characters per variant

For each variant, note whether it leads with the keyword, a question, or a value proposition.
```

**Usage Notes**
- Always ask for 5 variants — you'll usually combine elements from two or three
- Claude sometimes exceeds 155 characters; paste the output into a character counter before using
- Keyword-leading variants tend to perform better for informational queries; question-led ones work for comparison queries

## 3. Long-Form → Short-Form Repurposing

**The Prompt**
```
I have a long-form article and need it repurposed into shorter content formats. Based on the article below, produce:

1. A 250-word summary suitable for a newsletter
2. Three standalone pull quotes (1–2 sentences each) that work out of context
3. Five social post hooks (one sentence each) based on key insights

Article:
[paste article]

Do not use phrases like "In this article" or "The author argues." Write as if the insights are your own observations.
```

**Usage Notes**
- The "do not use phrases like..." instruction prevents the output sounding like a summary rather than content
- Pull quotes are useful for social media, slide decks, and email — save them in a swipe file
- Run this every time you publish long-form content; it takes 2 minutes and extends content lifespan significantly

## 4. Content Gap Identifier

**The Prompt**
```
I'm building a content strategy for [brand/publication] targeting [audience]. Based on the following list of existing content topics, identify:

1. Five topics we haven't covered that our audience likely searches for
2. Three existing topics we could go deeper on
3. Two content formats we're underusing for this audience

Existing content topics:
[paste list]

Audience description: [e.g. Singapore-based food enthusiasts who dine out 3–5 times per week]
```

**Usage Notes**
- The output quality depends heavily on how specific your audience description is — be precise
- Use this quarterly rather than one-off; content gaps shift as trends and search behaviour change
- Claude's format suggestions (point 3) are often obvious — push back with a follow-up: "What specific formats would suit this audience that most food media doesn't use?"
